# Quick Start
1. Download [NodeJS](https://nodejs.org/en/download/)
2. Install pnpm:
```bash
npm i -g pnpm
```
3. Go to:
```bash
cd App/Frontend/ReactClient
```
4. Install all dependencies:
```bash
pnpm i
```
5. Run in dev mode:
```bash
pnpm dev
```
6. Run in prod mode:
```bash
pnpm build
```
